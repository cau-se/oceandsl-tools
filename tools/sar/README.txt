Stand 15.8.2022:

Zum benutzen des bsc.dataflow packages muss unter folgendem Pfad die sich im diesen Verzeichnis befindene ZIP entpackt werden:

Pfad: '${PATH_TO_USER_DIR}\.m2\repository\net\kieker-monitoring\kieker\'
Zip: 'kieker-2.0.0-SNAPSHOT'

Diese Aktion wird überflüssig sobald die Naming Convention "AggregatedStorageAccess -> StorageDataflow" und das
zusätzliche Mapping angepasst wurde. Außerdem braucht es einen Release der neuen Klasse: 'OperationDataflow" mit zugehörigem
Mapping und der Implementation
